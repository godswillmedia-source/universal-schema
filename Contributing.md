# Contributing to Universal Schema

We welcome contributions to Universal Schema!

## How to Contribute

### Protocol Improvements

1. Open an issue describing the proposed change
2. Discuss with community
3. Submit pull request with:
   - Updated specification
   - Examples showing usage
   - Rationale for change

### Documentation

- Fix typos
- Add examples
- Clarify specifications
- Translate to other languages

### Reference Implementations

Share your Universal Schema implementation:
- Link to your repository
- Describe your use case
- Add to examples directory

## Guidelines

- Keep protocol simple
- Maintain backward compatibility when possible
- Provide clear examples
- Document breaking changes

## Questions?

Open a discussion in [GitHub Discussions](https://github.com/godswillmedia-source/universal-schema/discussions)
